# 🤖 BMO Robot – Calmi
**Type:** Hardware Graduation Project  
**Field:** Robotics & Embedded Systems  
**Department:** Computer Engineering – An-Najah National University  

An interactive educational robot designed to support children's learning, engagement, and safety through intelligent interaction and real-time parental monitoring.

---

## 🎯 Project Overview

BMO Robot is a hybrid embedded robotic system that combines:

- 🧠 Raspberry Pi 4 (High-level processing & AI interaction)
- ⚙️ Arduino Mega 2560 (Low-level hardware control)
- 📷 Computer Vision & Emotion Detection
- 🎮 Interactive learning activities
- 👨‍👩‍👧 Parental monitoring system
- 🍬 Reward-based mechanism

The robot integrates sensors, actuators, and AI modules to create a safe, child-friendly educational assistant.

---

## ✨ Key Features

### 👶 Child Mode
- 🎮 Hide and Seek game (Face + Thermal detection)
- 🃏 Learn with RFID Cards
- 😊 Emotion Recognition (Happy, Sad, Angry…)
- 🎤 Voice interaction using ChatGPT API
- 🍬 Reward system (Controlled candy dispensing)

### 👨‍👩‍👧 Parent Mode
- 🔐 Secure login system
- 📊 Real-time emotion monitoring
- 🌡 Environmental monitoring (Temperature, Humidity, Gas)
- 🚨 Emotion alerts for negative states
- 🎛 Movement & reward control
- ⏳ Daily playtime limitation

---

## 🧠 AI & Software Technologies

- Python 3
- OpenCV (Face Detection)
- TensorFlow & Keras (Emotion Recognition CNN)
- Whisper (Speech-to-Text)
- OpenAI ChatGPT API (Natural Language Interaction)
- Raspberry Pi OS

## 🔧 Hardware Components

- Raspberry Pi 4
- Arduino Mega 2560
- Ultrasonic Sensors (HC-SR04)
- RFID Reader + NFC Cards
- AMG8833 Thermal Sensor
- Temperature & Humidity Sensor
- Gas Sensor
- IR Safety Sensor
- DC Gear Motors + Wheels
- L298N Motor Driver
- Servo Motor (Candy Control)
- Limit Switches (Drawer Safety)
- USB Camera
- Microphone & Speaker
- Lithium Battery + BMS Module
- Tablet Display Interface

---

## 🏗 System Architecture

The robot follows a **Hybrid Architecture**:

- 🔹 Raspberry Pi → AI processing, emotion detection, voice interaction
- 🔹 Arduino Mega → Real-time motor & sensor control
- 🔹 Wireless communication → Parent monitoring system
- 🔹 Tablet interface → Child-friendly interaction
